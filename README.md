# 機能
レストラン検索。コメント投稿webアプリケーション。

# 特徴
- ### インフラ面
  - EKSで構築。Terraform使用。
  - 開発minikube、skaffold
- ### バックエンド面
  - GraphQL(Rails)
  - 認証(devise)
  - 外部API（リクルートAPI）を利用。
- ### フロントエンド面
  - Next.js、react-query、axiosを使用
